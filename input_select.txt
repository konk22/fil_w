Пример создания списка типов филамента:

input_select:
	name: Типы филамента
	options:
	  - "FDplast 2500/350"
	  - "FDplast 1000/170"
	  - "ABS Maker 1000/200"
	icon: mdi:factory

Добавить необходимо в файл configuration.yaml, после блока default_config.
Так же можно добавить через веб-интерфейс, в разделе: Устройства и службы -> Вспомогательное -> Создать вспомогательный объект -> Список
В случае добавления списка с помощью добавления в файл configuration.yaml, возможность редактирования списка через веб-интерфейс будет не доступна (Список будет в режиме чтения в разделе "Создать вспомогательный объект")
